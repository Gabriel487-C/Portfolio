Gerador De Ticket

Este projeto consiste em um gerador de tickets personalizado,
criado com HTML, CSS e JavaScript puro. O usuário preenche um formulário

Conversor de Dolar Para Real

Este projeto é um conversor de moedas desenvolvido com HTML, CSS e JavaScript,
integrando uma API de câmbio em tempo real para obter a cotação atual do dólar.
O usuário digita o valor em dólares e o sistema retorna automaticamente o equivalente
em reais, com base na taxa de câmbio mais recente.
